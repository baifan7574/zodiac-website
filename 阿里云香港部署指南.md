## 📋 第二步：购买/试用ECS服务器

### 步骤2.1：进入ECS控制台
1. 登录阿里云控制台
2. 在顶部搜索框输入：`ECS`
3. 点击 **"云服务器ECS"**

### 步骤2.2：创建实例
1. 点击 **"创建实例"** 或 **"立即购买"**
2. 开始配置

### 步骤2.3：基础配置

#### 1. 付费模式
- 选择：**"包年包月"** 或 **"按量付费"**
- 新用户建议：**"按量付费"**（可以随时停止，按小时计费）

#### 2. 地域和可用区
- **地域**：选择 **"中国（香港）"**
- **可用区**：选择任意可用区（例如：香港可用区A）

#### 3. 实例规格
- **规格族**：选择 **"共享计算型"** 或 **"突发性能实例"**
- **实例规格**：选择最便宜的配置
  - 例如：`ecs.t6-c1m1.large`（1核1GB）
  - 或：`ecs.t5-lc1m1.small`（1核1GB）
- **说明**：初期1核1GB足够使用，后续可以升级

#### 4. 镜像
- **操作系统**：选择 **"Ubuntu"**
- **版本**：选择 **"Ubuntu 20.04 64位"** 或 **"Ubuntu 22.04 64位"**

#### 5. 存储
- **系统盘**：选择 **"高效云盘"**
- **容量**：**40GB**（最低，足够使用）
- **数据盘**：不需要（初期不需要）

### 步骤2.4：网络和安全组

#### 1. 网络
- **网络类型**：选择 **"专有网络VPC"**
- **如果没有VPC**：系统会自动创建
- **公网IP**：选择 **"分配公网IPv4地址"**（必须！）

#### 2. 带宽
- **带宽计费方式**：选择 **"按固定带宽"** 或 **"按使用流量"**
- **带宽峰值**：选择 **"1Mbps"**（最低，足够使用）
- **说明**：初期1Mbps足够，后续可以升级

#### 3. 安全组
- **选择安全组**：选择 **"新建安全组"**
- **安全组规则**：系统会自动添加常用端口
  - SSH（22端口）
  - HTTP（80端口）
  - HTTPS（443端口）

### 步骤2.5：系统配置

#### 1. 登录凭证
- **登录方式**：选择 **"自定义密码"**
- **用户名**：`root`（默认）
- **密码**：设置一个强密码（至少8位，包含大小写字母、数字、特殊字符）
- **确认密码**：再次输入密码
- **重要**：务必保存这个密码！

#### 2. 实例名称
- **实例名称**：输入 `zodiac-website-server`
- 或您喜欢的任何名称

#### 3. 其他设置
- **资源组**：保持默认
- **标签**：不需要

### 步骤2.6：确认订单
1. 检查所有配置
2. 查看费用（按量付费通常很便宜，每小时几毛钱）
3. 勾选 **"《云服务器ECS服务条款》"**
4. 点击 **"确认订单"** 或 **"立即创建"**

## 📋 第三步：获取服务器信息

### 步骤3.1：等待实例创建
- 通常需要 1-2 分钟
- 在ECS控制台可以看到创建进度

### 步骤3.2：记录服务器信息
创建完成后，记录以下信息：

2. **用户名**
   - 默认：`root`

3. **密码**
   - 您刚才设置的密码

## 📋 第四步：配置安全组（开放端口）

### 步骤4.1：进入安全组
1. 在ECS控制台，点击实例名称
2. 进入实例详情页
3. 点击 **"安全组"** 标签

### 步骤4.2：添加入站规则
需要添加以下端口：

#### 规则1：HTTP（端口80）
- **规则方向**：入方向
- **授权策略**：允许
- **优先级**：1
- **协议类型**：自定义TCP
- **端口范围**：80/80
- **授权对象**：0.0.0.0/0
- **描述**：HTTP

#### 规则2：HTTPS（端口443）
- **规则方向**：入方向
- **授权策略**：允许
- **优先级**：1
- **协议类型**：自定义TCP
- **端口范围**：443/443
- **授权对象**：0.0.0.0/0
- **描述**：HTTPS

#### 规则3：后端API（端口3000）
- **规则方向**：入方向
- **授权策略**：允许
- **优先级**：1
- **协议类型**：自定义TCP
- **端口范围**：3000/3000
- **授权对象**：0.0.0.0/0
- **描述**：Backend API

## 📋 第五步：测试SSH连接

### Windows用户

#### 方法1：使用PowerShell
1. 打开PowerShell
2. 输入命令：
   ```bash
   ssh root@您的公网IP
   ```
   例如：
   ```bash
   ssh root@47.xxx.xxx.xxx
   ```
3. 输入密码（输入时不会显示，直接输入后按回车）
4. 如果连接成功，会看到欢迎信息

#### 方法2：使用PuTTY（推荐）
1. 下载PuTTY：https://www.putty.org/
2. 打开PuTTY
3. 填写信息：
   - **Host Name**：您的公网IP
   - **Port**：22
   - **Connection type**：SSH
4. 点击 **"Open"**
5. 输入用户名：`root`
6. 输入密码

### Mac/Linux用户
1. 打开终端
2. 输入命令：
   ```bash
   ssh root@您的公网IP
   ```
3. 输入密码

## ⚠️ 常见问题

### Q1: 找不到"创建实例"按钮？
**A**: 确保您已经登录，并且在ECS控制台页面。

### Q2: 没有看到"中国（香港）"选项？
**A**: 在"地域"下拉菜单中查找，或者搜索"香港"。

### Q3: 费用是多少？
**A**: 
- 按量付费：1核1GB大约每小时0.1-0.2元
- 包年包月：1核1GB大约每月30-50元
- 新用户有免费试用额度

### Q4: 无法SSH连接？
**A**: 
- 检查安全组是否开放了22端口
- 检查公网IP是否正确
- 检查密码是否正确

### Q5: 如何停止服务器节省费用？
**A**: 
- 在ECS控制台，选择实例
- 点击 **"停止"**
- 停止后不收费（但会保留IP，可能产生少量费用）
